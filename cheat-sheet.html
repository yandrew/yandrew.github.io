JavaScript Prototypes

Preface

This is a cheat sheet relating ruby syntax to JavaScript syntax

What is a prototype?

A prototype in Javascript is essentially a Class in Ruby. All objects in Javascript have prototypes. It provides a way to access methods / properties saved in objects that previously defined.

Defining prototypes (Defining Classes)

JavaScript
function Greeting() {
   this.message = "Greetings!";
}
Ruby
class Greeting
   attr_accessor :message
   def initialize
      @message = "Greetings!"
   end
end

Creating An Object

JavaScript
var hello = new Greeting();
Ruby
puts hello.message

Changing properties(attributes)

JavaScript
hello.message = "Hello!";
Ruby
hello.message = "Hello!"

Adding properties(attributes) to the protoype (class)

JavaScript
Greeting.prototype.alternative = "Hi!"
Ruby
There isn't a practical way to add attributes
to a class after it has been defined.
